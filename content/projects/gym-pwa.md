# Pengalaman Mobile-First untuk Pusat Kebugaran

Proyek ini adalah platform manajemen gym komprehensif yang menjembatani kebutuhan member dan staf operasional (admin/pelatih). Platform ini dibagi menjadi dua modul antarmuka utama.

## 1. Aplikasi Mobile PWA (Untuk Member)
Saya mengimplementasikan teknologi **Progressive Web App (PWA)** agar member dapat menginstal aplikasi langsung dari browser layaknya aplikasi *native* tanpa perlu lewat App Store/Play Store.
Fitur member meliputi:
- Kartu member digital dengan QR/Barcode.
- Katalog dan pemesanan paket keanggotaan.
- Jadwal latihan harian.
- Integrasi *payment gateway* untuk kemudahan pembayaran.

## 2. Admin Panel Terpusat (Untuk Staf & Pelatih)
Di sisi administratif, saya membangun dasbor web desktop yang bertenaga untuk:
- Mengelola jadwal latihan kelas secara dinamis.
- Manajemen siklus paket keanggotaan (aktif/kedaluwarsa).
- Pembuatan laporan operasional dan keuangan mendalam.

Menggunakan **Nuxt.js** dan **Tailwind CSS**, proyek ini sangat menonjolkan kemampuan saya dalam merancang arsitektur aplikasi yang responsif, cepat, dan berorientasi pada kemudahan akses di berbagai *device*.
